Welcome!

Welcome to the page for the extended time calculator. This tool is designed to help teachers calculate the time a student with a disability actually needs to use for testing. This tool is simply one tool to be used to collect one piece of data. Please read the README file for more information.

Do you have administrator privileges on your computer? If Yes continue below. If No, please click here.

 

If you have administrative privileges (you can freely install software on your computer)

Please follow these installation instructions, the program will not install if you do not.

Installation Instructions

  1. Click the "Download" button below.

    2. On Chrome

    1. Look to the bottom left corner of the chrome window. Click the caret symbol (^) on the file name, **DO NOT CLICK THE FILE**.
    2. Click *Show in folder*

      3. Once inside the downloads folder

    3. Right click the setup file
    4. Click *Run as administrator*
    5. If on Windows 10 a blue window may pop up. Click *more info* and click *run anyway*.
    6. Click yes on all pop-ups that you trust the source of this program (I hope you do!)
    7. Follow the setup procedure

    On Firefox

    1. Look to the top right corner. Click on the down arrow with a line under it (it should be blue).
    2. Click on the folder next to the name of the application setup file.

      3. Once inside the downloads folder

    3. Right click the setup file
    4. Click *Run as administrator*
    5. If on Windows 10 a blue window may pop up. Click *more info* and click *run anyway*.
    6. Click yes on all pop-ups that you trust the source of this program (I hope you do!)
    7. Follow the setup procedure

    Unless you change the install location this will install in *Program Files* or *Program Files (x86)*.   The name of the folder will be _**Extended Time Calculator vN.N(where N is the version number)**_.   This is the folder where you will find the *README*, and the *Student names* file.   The *database* file will be found in your *AppData/Roaming/Extended_Time_Calc* folder, which may be hidden.   The README file also pops up on install. _**It is very important that you read this file**_, it has instructions for use that are vital to the program not crashing.     Please **READ** the README.txt file in the program folder.
